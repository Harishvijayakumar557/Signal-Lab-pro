📉 Signal Lab Pro (Master CRO & Audio Lab)

An advanced, interactive **Signal Visualization & Audio Processing Suite**. This project goes beyond basic wave generation by incorporating real-time audio capture and playback capabilities.

💎 Key Features & Highlights

* 🎤 Live Voice Analysis: Real-time microphone input integration to visualize your own voice as a waveform.
* ⏺️ Record & Playback: Built-in functionality to capture live audio and play it back instantly for analysis.
* ⚡ Dual-Channel Precision: Independent controls for Channel 1 (Cyan) and Channel 2 (Yellow) to compare signal behavior.
* 🌀 Waveform Synthesis: Switch instantly between **Sine**, **Square**, and **Triangle** waves.
* Dynamic Manipulation: **Amplitude & Phase:** Real-time gain adjustment and X/Y axis positioning.
* Cyber-Tech UI: A futuristic "Cyberpunk" dark-mode interface for an authentic lab feel.

🛠️ Technical Capabilities

* Web Audio API: Used for high-fidelity microphone capture and audio processing.
* 60 FPS Canvas Rendering: Ultra-smooth signal animations using the HTML5 Canvas API.
* Low Latency: Optimized for real-time interaction without lag.
